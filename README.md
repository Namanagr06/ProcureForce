# ProcureForce

## Overview

ProcureForce is an enterprise procurement management application built on Salesforce Platform.

The application is inspired by SAP Materials Management (SAP MM) and is designed for organizations that require a configurable procurement lifecycle including Purchase Requisitions, RFQs, Supplier Quotations, Purchase Orders, Approvals, and Supplier Management.

---

## Project Goals

- Build an enterprise-grade Salesforce application.
- Follow Salesforce Well-Architected principles.
- Follow Salesforce DX development practices.
- Maintain complete source control using Git and GitHub.
- Produce enterprise documentation.
- Demonstrate real-world Salesforce architecture.
